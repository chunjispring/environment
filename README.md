## 简介
     *生命在于折腾，时间却不能浪费。*
>在重装系统后，跑一下脚本，设置一下使用环境（常用环境变量、alias等），安装一些常用软件（conky、openssh等），大大节省时间。

### 使用说明
下载此分支下的所有代码，然后运行安装脚本

     git clone https://github.com/chunjispring/environment.git
    
*  Linux类的系统（此脚本理论上支持debian系所有系统，包括ubuntu、deepin等）

      ./linux_setupenv.sh

*  MacOS系统

      ./mac_setupenv.sh
